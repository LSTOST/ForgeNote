## 3. 新布局（三区 + 顶栏，底栏取消）

```
┌────────────────────────────────────────────────────────┐
│ 顶栏: ⌂ | ◧ | 当前内容 · <标题>          <草稿保存状态> │
├──────┬──────────────────────────────┬──────────────────┤
│ 左栏  │ 中区（内容 + 主操作一条线）      │ 右栏（设置）       │
│ 导航  │                              │                  │
│      │ Stage 0: 想法输入              │ 状态 Badge（唯一） │
│ 新写  │   [生成内容框架]（输入框下方）   │ 写作顺序（可编辑）  │
│ 最近  │ Stage A: 框架卡片              │ 待确认            │
│ 选题  │   [生成正文]（框架列表下方）     │ 输出语言          │
│ 看板  │ Stage B: 正文编辑              │ 账号分析摘要       │
│      │   ── 产出区（正文之后）──        │                  │
│ 页脚  │   平台选择 chips + [生成]       │                  │
│ 身份  │   平台版本记录（常驻列表）       │                  │
└──────┴──────────────────────────────┴──────────────────┘
```

分阶段主操作（一屏一个 brand 按钮）：

| 阶段 | 主操作 | 位置 |
|---|---|---|
| Stage 0 | 生成内容框架 | 输入框正下方 |
| Stage A | 生成正文 | 框架列表末尾 |
| Stage B | 生成小红书版本（文案+卡片提示词）/ 生成 X 版本 / 生成卡片提示词 | 产出区（正文之后） |

产出区细则：

- 平台 chips（小红书 / X / 卡片提示词）+ 主按钮 + 错误提示，替代原底栏。
- 「平台版本记录」从右栏移入产出区（产物属于内容流，不是设置）；点击打开 overlay 回看/复制。
- 重新生成正文入口保留在正文标题行（次级链接样式）。
