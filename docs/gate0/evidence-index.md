# Gate 0 证据归档索引

所有 Gate 0 真实路径证据必须归档到 `docs/acceptance/`，文件命名：

- `gate0-week-1.md`
- `gate0-week-2.md`
- `gate0-week-3.md`
- `gate0-week-4.md`
- `gate0-four-week-summary.md`

每周证据必须包含：

- 日期范围和账号。
- 至少一条真实内容任务。
- 选题来源：radar / own_idea / recipe。
- 是否发布。
- 是否回填表现。
- 是否写回 account memory。
- ChatGPT fallback 次数和原因。
- Owner 主观判断：下周是否还愿意先打开 ForgeNote。

禁止：

- 用 Codex 代码检查替代 Owner 真实路径。
- 用 mock 数据冒充真实发布。
- 只贴 dashboard 数字，不解释本周是否真的离不开。
- 把正文全文塞进 evidence；只贴链接、截图说明、id、摘要。
