## 0. 一句话

**ForgeNote 首切口 = 跑通 Owner「独居」内容生意的周更生产：选题进来 → 小红书图文成品出去 → 挂上手册/清单链接 → 发布后回填。**

单语言（中文）、先单平台（小红书），公众号长文是切片 2。双语 / X / 多平台广度 / 灵感库维持冻结（D-13）。

## 1. 事实依据（Owner 真实工作流，2026-07-12 确认）

- 业务：围绕「独居」的两条生产线——**手册线**（批量项目：系统手册 + 商品详情页 + 公众号/豆瓣长文介绍）与**选题线**（周常循环：小红书图文 + 公众号长文，嵌工具清单引流、挂手册售卖）。
- 变现三条腿：手册售卖、网盘拉新转存收益、清单引流养流量池。
- 痛点排序（Owner 自报）：小红书图文成稿、长文类生产、资产关联管理三者并列；**选题与规划不痛**（已有成熟选题体系）。
- 节奏现状：尚未稳定——工具的隐含目标之一是把节奏稳下来。
- 定位拍板：ForgeNote 首切口 = 独居生意的生产台，不追求通用性。
