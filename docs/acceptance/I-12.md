## 鉴权 / 错误码

- 未登录 → `AUTH_REQUIRED`（401）。
- 非法 JSON / range 非枚举 / `publishedAt` 非 ISO / note 超 500 → `VALIDATION_FAILED`（400）。
- 至少需一项表现字段，否则 `VALIDATION_FAILED`（400，在鉴权之后）。
- 他人 / 不存在 / 非法 id → `SESSION_NOT_FOUND`（404，依赖 RLS + `user_id`，**不泄露存在性**，不返回 `PERMISSION_DENIED`）。
- 鉴权顺序：解析 body → schema → 鉴权 → 路径 UUID → 至少一项 → RLS 更新。
- 成功后 best-effort 记录 `usage_events.performance_filled`（仅记字段名，不存复盘正文）。
