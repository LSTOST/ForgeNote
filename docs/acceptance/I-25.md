## 实现内容（按 ia.md §2 + design-system-baseline）

- **引导态 / 工作台态分离（progressive disclosure，ia §2 / S-08）**：
  - `status === "idle" && !sessionId` → 只渲染居中的当前任务输入 + 「我会替你补：受众·平台·形式·目标·风格」+ 主操作「先确认方向」；**不渲染左/右/底空面板**（消除空工作区压迫）。
  - 进入 `review` 或有 `session` → 渲染四区工作台。
- **桌面四区（≥xl）**：`xl:grid-cols-[236px_1fr_356px]` + 两行显式放置：
  - 左（col1，跨两行）：账号上下文——ForgeNote/内容工作台、导航（当前任务 / 配方库 / 账号偏好）、账号记忆 & 内容资产**中性说明**（不伪造资产库数据）。
  - 中上（col2 row1）：当前任务输入（IdeaInput）。
  - 中下（col2 row2）：内容方案结果（OutcomePanel）或安静占位。
  - 右上（col3 row1）：输出语言/表达偏好 + I-24 假设 chip（DirectionPanel，`compact`，未改数据结构/行为）。
  - 右下（col3 row2）：保存配方（RecipePanel）。
  - 底（全宽）：本次 session / 复用 / 表现连续性条。
- **移动单列优先级（ia §9 / 反 patch §10）**：`flex-col` + `order-*`，桌面 `xl:order-none` + 显式 grid 放置。移动顺序：任务输入 → 输出偏好+假设 → 内容方案 → 保存配方 → 底部连续性；左栏导航在移动端隐藏（走全局 TopNav），不复制四栏。
- **暖纸感基线对齐（§0/§1/§4/§6，替换旧 stone/amber + 深绿 `#18322f` 侧栏）**：surface `#FBF7F0` / raised `#FFFDF9` / border `#E3D8C7` / ink `#2A2320`·`#6f6253`·`#9a8e7c` / accent `#B5562B` / 成功 `#2F7A4F` / 焦点环 `#B5562B`/.28。每屏唯一主操作（引导态=先确认方向，工作台=生成内容方案）。
- **copy**：新增 `workspace.*`（左栏/导航/占位/连续性/引导补全），en/zh key 对齐（typecheck 保证）；`shell.*`/`direction.*`/`forge.*` 复用。
- **IdeaInput 轻量适配**：输入框与主按钮换到暖纸 token（border `#E3D8C7`/bg `#FFFDF9`/accent `#B5562B`）；未改其逻辑。
