## 结论

- Scope not touched: `/api/forge`, prompt, AI contract, DB, RLS, `profile_preferences`, auth, analytics, whole `/forge` redesign, untracked `原型图/`.

## 实现验收

- Three direction assumptions render a compact chip containing dimension + current value.
- Confidence labels use `确定` / `推断` / `拿不准`; shape/border carries the state instead of adding more color noise.
- Rationale disclosure uses `我这样判断是因为：...`; `拿不准` or missing-rationale assumptions default open, normal inferred assumptions default folded.
- Editing one assumption sets `state="edited"`, `source="manual"`, `confidence="high"`, updates the count to `已确认 1/3`, and exposes `恢复默认`.
- Rejecting an assumption marks it `dismissed`; generation continues to submit only non-dismissed assumptions, matching the previous request boundary.
- Restoring an edited/rejected assumption rebuilds the current default from the current idea, optional prior post, and saved profile assumptions.
- Mobile layout uses wrapped chips/buttons and keeps the all-rejected notice above the disabled generate action to avoid horizontal squeeze.
